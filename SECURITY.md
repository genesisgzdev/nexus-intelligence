# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 4.0.x   | Yes       |
| < 4.0   | No        |

## Reporting Vulnerabilities

Report security vulnerabilities privately to genzt.dev@pm.me

Include:
- Vulnerability type
- Affected versions
- Reproduction steps
- Proof of concept

## Security Best Practices

- Store API keys in .env file only
- Never commit credentials
- Use HTTPS proxies
- Implement rate limiting
- Monitor network traffic

## Legal Use Only

This framework must only be used for authorized security assessments.
