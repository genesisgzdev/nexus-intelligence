# Security Policy (Nexus Intelligence)

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| 1.1.x   | :x:                |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of Nexus Intelligence seriously. If you find a security vulnerability, please report it privately to:

**Security Team:** security@nexus-intel.local (or your designated private channel)

Please include:
- A description of the vulnerability.
- Steps to reproduce.
- Potential impact.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
